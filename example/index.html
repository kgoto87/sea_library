Sea
- JS and CSS3 Libraby for Super Easy Animation -

Sea is the easiest(ish) way to make your website more attractive!

How To Use

Set it

STEP

Download the Sea library and splace it in your project directory.

STEP

Set the code below to the HEAD of your webpage

<script src="path/to/sea/sea.js"></script>

STEP

Set class of elements as follows and let the magic happen!
